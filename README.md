# BeatJam

A multiplayer music rhythm game built in Godot Engine, featuring a custom controller using Raspberry Pi.

## Overview

BeatJam is a fun and engaging rhythm game that using a custom controller adressable via Raspberry Pi. Players can select songs, hit beats in time with the music, and compete for high scores.

## Features

- **Custom Controller**: A unique physical controller built with Raspberry Pi for an immersive gaming experience.
- **Multiplayer Mode**: Cooperate or compete with friends in local multiplayer mode.
- **Song Selection**: Choose from a variety of songs to play.
- **Scoring System**: Each player has their own score, and the game keeps track of high scores.
- **Visual Effects**: Engaging visual effects that sync with the music.
- **Simple UI**: Easy-to-navigate user interface for selecting songs and viewing scores.
- **Selfmade Assets**: All game assets, including music and graphics, are created by the developer.
